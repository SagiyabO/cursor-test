# CLI Sum Calculator

This project contains a simple Python command-line script that prompts for two numbers (X and Y), calculates their sum, and prints a friendly message with the result.

## Requirements
- Python 3.13 or compatible Python 3 version

## Usage
1. Open a terminal in this directory.
2. Run the script:
   ```bash
   py test1.py
   ```
   or
   ```bash
   python test1.py
   ```
3. Enter numeric values for X and Y when prompted. The script will display the sum using a nicely formatted output that avoids floating-point precision artifacts.

## Project Structure
- `test1.py`: Main entry point for the CLI calculator.
- `README.md`: Project documentation (this file).
- `requirements.txt`: Placeholder for project dependencies (empty by default).
